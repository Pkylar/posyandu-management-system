## 🚀 Cara testing:

1. **Buka http://localhost:3001 atau 3002**
2. **Login** dengan username/password apapun
3. **Ke menu Data Balita**
4. **Klik tombol "Detail"** pada salah satu balita
5. **Lihat halaman detail** dengan:
   - Data diri lengkap (nama, gender, tempat lahir, dll)
   - Data keluarga (ayah, ibu, alamat, RT/RW)
   - Riwayat penimbangan (tabel)
   - Grafik berat badan (line chart hijau)
   - Grafik tinggi badan (line chart biru)

## 🔧 Jika masih bermasalah:
